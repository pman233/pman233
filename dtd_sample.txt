Usage:

    This file must be included in a DOCTYPE statement before it can be parsed.

  VCESInn
  =======
    To create a DTD for creation of service information elements as well as
    all required qualifying information, append the following statement to
    the beginning of the file:
    
       <!DOCTYPE vcesi [   

       and the associated "]>" to the end of the file.

  Example of an XML document instance:

     <!DOCTYPE vcesi PUBLIC "-//Volvo Construction Equipment//DTD VCESInn//EN" 
                            "\path\vcesinn.dtd" 
     [
     ..... entity declarations .....
     ] >
     <vcesi>
     .... 
     </vcesi>

    Where nn is the version no of the DTD
